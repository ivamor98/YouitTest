# Тест задача
## Условие
Дана строка s (1 ≤ |s| ≤ 10^6), состоящая из латинских строчных букв.
Найдите максимальную по длине последовательность подряд идущих символов, где буквы составляют подстроку вида "youit" (возможно, повторяющуюся, при этом не обязательно полную, но обязательно начинающуюся с символа 'y').
Пример: abcyouityouzxc содержит последовательность "youityou" длины 8, подходящую под указанные условия.

В первой строке вводится количество независимых тестов t (1≤t≤10^5). В следующих t строках вводится строка  (1 ≤ |s| ≤10^6).
Гарантируется, что сумма длин строк во всех t тестах не превышает 10^6 символов. 

## Тестовые значения
## Input
9
youityou
bbbbbybbb
xyz
youiyouit
t
y
youitxyouityouit
yoyoyoy
abcyouityouityouityouityouityouityoui
## Output
8
1
1
5
0
1
10
2
34
